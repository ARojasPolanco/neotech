# Neo Tech — Brand Spec

## Color Tokens (OKLch + Hex)

### Primary
- `--bg`: #FAFAFA → oklch(98% 0.002 100)
- `--surface`: #F5F5F5 → oklch(96% 0.003 100)
- `--fg`: #0A0A0A → oklch(13% 0.01 100)
- `--muted`: #6B6B6B → oklch(48% 0.01 100)
- `--border`: #E0E0E0 → oklch(92% 0.005 100)

### Accent
- `--accent`: #A8E05F → oklch(82% 0.18 130)
- `--accent-dark`: #8BC34A → oklch(74% 0.18 130)

### Status
- `--success`: #22C55E → oklch(72% 0.19 145)
- `--warning`: #F59E0B → oklch(82% 0.16 85)
- `--error`: #EF4444 → oklch(63% 0.24 25)

## Typography

- **Display/Headings:** Barlow Condensed (800, 700, 600)
- **Body:** Inter (400, 500)

## Layout Rules

- Base unit: 4px
- Screen padding: 20px horizontal
- Card inner padding: 16px
- Between sections: 24px
- Safe area bottom (iOS): 34px

## Component Rules

- Primary button: full-width, #A8E05F background, dark text, radius 12px, height 52px
- Cards: radius 16px, surface background
- Inputs: radius 12px, surface background, border on focus
- Pill badges: radius 999px

## Brand Voice

- Confiable, rápido de navegar, mobile-first
- Verde lima como acento memorable, no protagonista visual
- Sin degradados llamativos ni múltiples colores saturados
